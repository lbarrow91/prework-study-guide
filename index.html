Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visable content shown to the user
  • Section elements are semantic elements that define a section in a document that contains thematic content
  • An element refers to the entire element from opening tag to closing tag
  • A tag refers only to what apppears inside the angle brackets
  • The element that holds all the other elements is known as the parent element - such as the BODY element
  • Elements that are nested inside other elements are called child elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • When a developer wants to style an element directly in the HTML file, they use inline CSS
  • Inline CSS is useful for quick and permanent changes but is less flexible because each inline style must be edited individually whenever you make a design change
  • Internal CSS style sheet can be applyed to a whole parent element such as 'Head' - the rules only apply to that page and a single change to the CSS rule will apply to all tagged page elements
  • An external CSS style sheet is preferred in most cases because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier
  • We use a 'link' element to link the external CCS style sheet within the HTML
  • Declarations contain two important components: the CSS property we want to apply and the value of the property
  • A margin property indicates how much space we want around the outside of an element - it is not visable on the page
  • The padding property adds space around the content inside an element inside the border
  • For the selector, a special wild card symbol, or *, applies the rule to all the elements visible on the page
  • We add the link to the external CSS style sheet to the 'head'section of the HTML code. The 'rel' attribute specifies the relationship between the current document and the linked document resource and the 'href' attribute specifies the location (URL)
  • The full stop preceding the class designates this as a class selector. Any HTML element assigned to the class will gain these property assignments from the CSS.

Git

  • git status: checks what branch we are currently on
  • git checkout -b brand-name: creates a new branch and switches to it
  • git branch: lists your branches
  • git checkout: Switches to another branch
  • git add -A: Adds changed files in the working branch (stored locally) to the staging area of Git
  • git commit -m "comment on what has been done": Commits changes which have been staged to the repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a programming language that allows users to interact with websites
  • One of its core features is variables, which store data types as values
  • JavaScript link is placed at the bottom of the HTML code as the browser reads code in the order it appears in the file
  • If the JavaScript loads first and it is supposed to affect the HTML that hasn't loaded yet, there could be problems
  • Variables are containers that store values
  • A semicolon at the end of a line indicates where a statement ends
  • Number and Boolean (True/False) variables don't need quote marks
  • You can add comments by enclosing them in /* or // - the browser ignores them
  • JavaScript is read from top to bottom, and order counts
  • 'if' statements allows us to interrupt control flow
  • An array is a single variable that is used to hold a group of data
  • In JavaScript the first item in any array has an index of 0
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array
  • A function is a set of instructions that tells the computer how to perform a certain task and only executes when called by name
  • A function's name is followed by parentheses ()
